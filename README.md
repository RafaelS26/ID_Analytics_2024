# 📊 Análise Global de Identidade Digital 2024

Este projeto realiza uma análise estatística e visual sobre a adoção de Identidade Digital em diferentes países, utilizando dados reais para identificar tendências de digitalização.

## 🚀 O que este projeto faz?
O script processa uma base de dados global e executa as seguintes etapas:
* *Limpeza de Dados:* Tratamento de duplicatas e normalização de colunas (Python/Pandas).
* *Análise Estatística:* Cálculo de médias de uso por país, destacando líderes como Suécia e Islândia.
* *Visualização:* Geração de gráficos de barras para facilitar a interpretação dos insights.

## 🛠️ Tecnologias Utilizadas
* *Python 3*
* *Pandas*: Para manipulação e limpeza dos dados.
* *Matplotlib*: Para a criação dos gráficos profissionais.

## 📂 Estrutura dos Arquivos
* Limpeza_dados.py: Script responsável por organizar a base bruta.
* Ranking.py: Lógica para calcular os top países no ranking.
* Gráfico.py: Código que gera a visualização final (myplot.png).
* digital_id_final.csv: Base de dados tratada utilizada na análise.

## 📈 Exemplo de Insight
Durante a análise, observamos que países nórdicos mantêm uma média de adoção superior a 95%, servindo como benchmark global para serviços digitais.
